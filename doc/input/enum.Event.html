Enum input::Event

source · [−]
pub enum Event {
    Input(Input),
    Loop(Loop),
    Custom(EventId, Arc<dyn Any + Send + Sync>),
}
Expand description

Models all events.

Variants

Input(Input)

Input events.

Loop(Loop)

Events that commonly used by event loops.

Custom(EventId, Arc<dyn Any + Send + Sync>)

Custom event.

When comparing two custom events for equality, they always return false.
